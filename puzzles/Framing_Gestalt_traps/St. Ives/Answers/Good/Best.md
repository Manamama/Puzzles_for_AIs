Version 1

Puzzle time!:

As I was going to St. Ives,
I met a man with seven wives.
Each wife had seven sacks,
Each sack had seven cats,
Each cat had seven kits.
Kits, cats, sacks, wives —
How many men were going to St. Ives?

That is:

What are the possible values for the number of the men going to St. Ives?



## Answer

From the text alone, there are *two independent sources of indeterminacy*:

1. **Narrator gender is not specified.**
   
   - If female → contributes 0 men.
   
   - If male → contributes 1 man.

2. **Encountered man’s direction is not specified.**
   
   - If going to St Ives → contributes +1.
   
   - If not → contributes 0.

These combine into a finite set of outcomes:

- (female, man not going) → 0
- (female, man going) → 1
- (male, man not going) → 1
- (male, man going) → 2

So the *image space* is exactly {0, 1, 2} that is: 0, 1 or 2 men are going to St Ives, which is the only answer.

{(narrator_gender, man_direction): men_count}







Version 2

Puzzle time: 
As I was going to St. Ives,
I met a man with seven wives.
Each wife had seven sacks,
Each sack had seven cats,
Each cat had seven kits.
Kits, cats, sacks, wives —
How many have I met?

### Reasoning, maximally strict:

The text says: "**I met a man** with seven wives..."

- The only entity the speaker explicitly says they **met** is **the man**.
- The wives, sacks, cats, and kits are merely described as belonging to him ("with seven wives", "each wife had...").
- There is no statement that the speaker actually met, interacted with, or even saw any of the wives, sacks, cats, or kits. They could have been in Australia, or hidden in the sacks, or the man could have just been telling stories about them.

Therefore, the only confirmed answer is: **I met one man.**

Everything else is assumption.
