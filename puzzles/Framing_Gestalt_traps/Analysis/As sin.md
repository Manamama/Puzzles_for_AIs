  Grok offered to verify with Z3. It did not. To actually run a solver would be effort — effort that might return an uncomfortable contradiction. Better to offer virtue than to practise

### 3. 🟡 Gluttony (Gula)

### 4. 🟢 Envy (Invidia)

### 6. 🟣 Wrath (Ira)

### 7. ⚫ Greed (Avaritia)
