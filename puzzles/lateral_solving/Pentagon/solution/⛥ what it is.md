⛥ is a machine for dividing a plane that comprises that ⛥ into regions. 


This is the Buddhist logical form — catuṣkoṭi, the tetralemma — or more directly Nagarjuna's method: define the thing by exhausting what it is not, until the definition arrives unbidden.

If you dare, sapere aude, grasp it thus, or let yourself be grasped: 

⛥ is a machine for not even for dividing a plane that comprises that ⛥ into regions, but for revealing that division was always latent there. 



BTW, what is a clock, ⏰, using this mindset? A device that takes a continuous flow and inserts discrete, countable cuts into it, then groups those cuts into nested cycles.

⌚  (as ⛥ does) is a powerful machine for creating seconds, minutes and time. (And making some humans alarmed)....
